# React Disney Plus App

TMDB API를 활용해 Disney Plus App을 만든 예제로 아래 명시된 기술을 활용했습니다.

- React
- React Router Dom
- Styled Components
- Swiper
- Axios
- Firebase Authentication / Hosting
